# Power BI Basics

 1. Overview

| Topic | Description |
|------|------------|
| Tool | Power BI |
| Purpose | Data visualization & reporting |
| Output | Dashboards & reports |

 2. Components

| Component | Description |
|----------|------------|
| Power BI Desktop | Report creation |
| Power BI Service | Sharing dashboards |
| Power Query | Data cleaning |
| DAX | Calculations |

 3. Workflow

| Step | Action |
|------|-------|
| 1 | Import data |
| 2 | Clean using Power Query |
| 3 | Build data model |
| 4 | Create visuals |
| 5 | Publish report |

 4. Data Modeling

| Concept | Description |
|--------|------------|
| Fact Table | Numeric data |
| Dimension Table | Descriptive data |
| Relationship | Connect tables |
| Star Schema | Best practice model |

 5. Visualizations

| Type | Use Case |
|------|----------|
| Bar Chart | Comparison |
| Line Chart | Trends |
| Pie Chart | Proportion |
| Card | KPIs |

 6. Filters & Slicers

| Feature | Purpose |
|--------|--------|
| Filters | Limit data |
| Slicers | Interactive filtering |




 Key Takeaways

- Power BI is used for dashboards  
- Data modeling is critical
