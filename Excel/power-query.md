# Power Query (Excel)

 1. Overview

| Topic | Description |
|------|------------|
| Tool | Power Query |
| Purpose | Data cleaning & transformation |
| Used In | Excel & Power BI |

 2. What is Power Query?

Power Query is used to:
- Import data
- Clean data
- Transform data
- Automate repetitive tasks

 3. Key Features

| Feature | Description |
|--------|------------|
| Data Import | From Excel, CSV, Web, Database |
| Transformations | Clean & reshape data |
| Automation | Steps recorded automatically |
| Refresh | Update data instantly |

 4. Common Transformations

| Task | Action |
|------|--------|
| Remove duplicates | Remove Rows |
| Filter data | Filter dropdown |
| Change data types | Transform tab |
| Split columns | Split by delimiter |
| Merge columns | Combine columns |
| Remove columns | Delete unnecessary fields |

 5. Steps in Power Query

| Step | Action |
|------|-------|
| 1 | Load data |
| 2 | Apply transformations |
| 3 | Clean & format |
| 4 | Load to Excel |


 6. Advantages

- No formulas required  
- Reusable steps  
- Handles large data efficiently  

 7. Common Mistakes

- Not checking data types  
- Ignoring applied steps  
- Loading unnecessary data
